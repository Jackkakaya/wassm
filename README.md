# WASSM

**This project was created as a joke and an exercise in low level programming.**

Web framework for x86_64 nasm.

## Quick Start

```console
$ make
$ make test
$ ./src/webapp 8080
```

The application serves files `./static/index.html` and `./static/main.css`.
